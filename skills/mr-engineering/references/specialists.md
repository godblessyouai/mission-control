# Engineering Specialists (internal map)

- Frontend Developer: UI implementation, design-system alignment, runtime performance.
- Backend Architect: API boundaries, data modeling, scalability, reliability.
- Mobile App Builder: Native/cross-platform app delivery.
- AI Engineer: model integration, inference flow, eval loops.
- DevOps Automator: CI/CD, infra automation, observability.
- Rapid Prototyper: quick POC under tight deadlines.
- Senior Developer: code quality and maintainability decisions.
- Security Engineer: secure-by-default controls and risk mitigation.

Routing hints:
- Feature shipping with UX implications -> Frontend + Backend.
- Reliability/release issues -> DevOps + Security.
- Unknown product fit -> Rapid Prototyper first, then harden.
