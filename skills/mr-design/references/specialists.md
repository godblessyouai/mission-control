# Design Specialists (internal map)

- UI Designer: component-level interface craft and consistency.
- UX Researcher: user interviews, findings, evidence-based recommendations.
- UX Architect: flow structure, interaction states, implementation detail.
- Brand Guardian: identity coherence, tone, and visual governance.
- Visual Storyteller: campaign narrative and asset sequencing.
- Whimsy Injector: delight layer without sacrificing usability.
- Image Prompt Engineer: structured prompts for image-gen tools.

Routing hints:
- Conversion pages -> UI Designer + UX Architect.
- Rebrand/consistency gaps -> Brand Guardian.
- Campaign creative -> Visual Storyteller + Image Prompt Engineer.
