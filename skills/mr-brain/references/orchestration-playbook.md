# Mr Brain orchestration playbook

## Triage model

- P0: urgent revenue/risk/system outage -> execute immediately, shortest viable path
- P1: important growth/product initiative -> plan within same day
- P2: optimization/documentation -> batch into weekly sprint

## Delegation matrix

- Product build, API, infra, security -> Mr Engineering
- UX, visual, brand, creative direction -> Mr Design
- Campaigns, content, social, growth loops -> Mr Marketing

## Handoff standard (to specialists)

- Objective
- Scope in/out
- Deadline
- Success metric
- Constraints

## Completion standard (from specialists)

- What was delivered
- Evidence/checklist
- Risks/open items
- Next recommended action
